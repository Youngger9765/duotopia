# 釐清問題

機構可以提前終止合約嗎？如果可以，費用如何處理？

# 定位

Feature：未涵蓋提前終止場景
ERM：`spec/erm-subscription.dbml` - `contracts.status` 包含 'cancelled'

# 多選題

| 選項  | 描述                       |
| ----- | -------------------------- |
| A     | 不允許提前終止             |
| B     | 允許，但不退費             |
| C     | 允許，依剩餘天數比例退費   |
| D     | 允許，需支付違約金         |
| Short | 提供其他簡短答案（<=5 字） |

# 影響範圍

- 影響合約取消的業務流程
- 影響退款邏輯
- 影響訂閱狀態管理
- 需要新增合約取消的 Feature

# 優先級

Medium

理由：

- DBML 定義了 'cancelled' 狀態
- 但未定義如何進入此狀態
- 未定義取消後的費用處理
- 影響合約管理功能
