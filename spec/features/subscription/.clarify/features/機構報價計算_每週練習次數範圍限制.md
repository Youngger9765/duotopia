# 釐清問題

報價計算機的「每位學生每週練習次數」是否有範圍限制？

# 定位

Feature：`spec/features/subscription/機構報價計算.feature` - 規則「計算預估年度點數」

# 多選題

| 選項  | 描述                       |
| ----- | -------------------------- |
| A     | 1-7 次（每天最多一次）     |
| B     | 1-10 次（合理範圍）        |
| C     | 1-20 次（寬鬆範圍）        |
| D     | 無限制，任意正整數         |
| Short | 提供其他簡短答案（<=5 字） |

# 影響範圍

- 影響報價計算功能的輸入驗證
- 影響 UI 的輸入範圍限制
- 影響點數預估的合理性檢查

# 優先級

Medium

理由：

- Feature 範例使用 3 次和 5 次
- 未定義最小值和最大值
- 影響使用者輸入驗證
- 不影響核心計算邏輯

---

# 解決記錄

- **回答**：A - 固定範圍（1~7 次/週），預設 3 次
- **更新的規格檔**：spec/features/subscription/機構報價計算.feature
- **變更內容**：新增「報價輸入條件」規則，明確定義每週練習次數範圍 1~7，預設 3，超出範圍顯示錯誤。
