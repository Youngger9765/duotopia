# 釐清問題

點數加購是否有頻率限制？例如：一天/一週最多加購幾次？

# 定位

Feature：`spec/features/subscription/點數加購.feature` - 規則「加購 10k 點數包」與「加購 50k 點數包」

# 多選題

| 選項  | 描述                           |
| ----- | ------------------------------ |
| A     | 無限制，隨時可加購             |
| B     | 一天最多加購 3 次              |
| C     | 一週最多加購 5 次              |
| D     | 只能在剩餘點數低於某閾值時加購 |
| Short | 提供其他簡短答案（<=5 字）     |

# 影響範圍

- 影響加購功能的驗證規則
- 影響 UI 的按鈕啟用/禁用邏輯
- 可能需要新增頻率限制的 Feature Rule

# 優先級

Low

理由：

- Feature 未提及頻率限制
- 一般來說點數加購應該隨時可用
- 不影響核心功能實作
- 可後續根據業務需求調整

---

# 解決記錄

- **回答**：A - 無限制，隨時可加購
- **更新的規格檔**：spec/features/subscription/點數加購.feature
- **變更內容**：新增「加購頻率限制」規則，明確定義無加購次數限制，隨時可加購。
