# 釐清問題

報價計算中「機構有 N 位英文老師」與簽約時「總教師數」的對應關係？

# 定位

- Feature：`spec/features/subscription/機構報價計算.feature` - 使用「機構有 N 位英文老師」計算贈額
- Feature：`spec/features/subscription/機構簽約與付款.feature` - 合約中的「總教師數」
- 兩者是否為同一數字？

# 多選題

| 選項  | 描述                                         |
| ----- | -------------------------------------------- |
| A     | 相同（機構英文老師數 = 合約總教師數）        |
| B     | 不同（機構可能有更多老師，但只訂閱部分名額） |
| C     | 機構英文老師數是參考，總教師數可調整         |
| D     | 機構英文老師數 ≥ 總教師數（不能超訂）        |
| Short | 提供其他簡短答案（<=5 字）                   |

# 影響範圍

- 影響報價計算的輸入參數定義
- 影響合約建立時的資料驗證
- 影響兩個 Feature 之間的資料一致性

# 優先級

High

理由：

- 「機構報價計算.feature」：機構有 10 位英文老師 → 贈送 1 位 → 總授權 11 位
- 「機構簽約與付款.feature」：機構有 10 位英文老師，購買 7 位，贈送 3 位，總授權 10 位
- 兩個範例的計算邏輯不一致
- 需釐清正確的計算方式
