# 釐清問題

訂閱到期後，剩餘點數如何處理？

# 定位

Feature：未涵蓋訂閱到期場景
ERM：`spec/erm-subscription.dbml` - Table `subscriptions` - 欄位 `remaining_points`

# 多選題

| 選項  | 描述                       |
| ----- | -------------------------- |
| A     | 清零（點數不可延續）       |
| B     | 保留（可在續約後繼續使用） |
| C     | 退款（依剩餘點數比例退費） |
| D     | 依合約類型不同而異         |
| Short | 提供其他簡短答案（<=5 字） |

# 影響範圍

- 影響訂閱到期處理邏輯
- 影響點數交易記錄（可能需要 'expire' 類型）
