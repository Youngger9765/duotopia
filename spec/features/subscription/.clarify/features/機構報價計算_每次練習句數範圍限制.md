# 釐清問題

報價計算機的「每次練習句數」是否有範圍限制？

# 定位

Feature：`spec/features/subscription/機構報價計算.feature` - 規則「計算預估年度點數」

# 多選題

| 選項  | 描述                       |
| ----- | -------------------------- |
| A     | 5-30 句（常見練習範圍）    |
| B     | 10-50 句（寬鬆範圍）       |
| C     | 1-100 句（極寬鬆範圍）     |
| D     | 無限制，任意正整數         |
| Short | 提供其他簡短答案（<=5 字） |

# 影響範圍

- 影響報價計算功能的輸入驗證
- 影響 UI 的輸入範圍限制
- 影響點數預估的合理性檢查

# 優先級

Medium

理由：

- Feature 範例使用 15 句和 20 句
- 未定義最小值和最大值
- 影響使用者輸入驗證
- 不影響核心計算邏輯
