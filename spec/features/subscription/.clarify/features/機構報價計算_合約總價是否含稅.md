# 釐清問題

機構簽約時，合約總價是否包含稅金？

# 定位

Feature：`spec/features/subscription/機構報價計算.feature` - 規則「計算一年約報價」與「計算兩年約報價」

# 多選題

| 選項  | 描述                             |
| ----- | -------------------------------- |
| A     | 未含稅（需另外計算營業稅）       |
| B     | 已含稅（總價即為最終金額）       |
| C     | 點數費用含稅，教師授權費用未含稅 |
| D     | 依機構類型不同（營利/非營利）    |
| Short | 提供其他簡短答案（<=5 字）       |

# 影響範圍

- 影響 `contracts.total_contract_value` 的計算公式
- 影響報價顯示的格式與說明
- 影響發票開立邏輯
- 可能需要新增稅金相關欄位

# 優先級

Medium

理由：

- Feature 計算總價時未提及稅金
- 台灣一般需要加計 5% 營業稅
- 影響最終金額計算
- 不影響核心比例關係

---

# 解決記錄

- **回答**：A - 合約總價已含稅，稅率固定 5%
- **更新的規格檔**：spec/features/subscription/機構報價計算.feature
- **變更內容**：新增「合約總價含稅」規則，明確定義合約總價已含稅，稅率固定 5%，報價顯示已含稅標註。
