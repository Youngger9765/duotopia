# 表達訓練模組術語詞彙表

本詞彙表定義 output-training 模組中使用的標準術語，確保規格文件的一致性。

## 核心概念

| 術語     | 英文                  | 定義                             | 使用範例             |
| -------- | --------------------- | -------------------------------- | -------------------- |
| 主題     | Topic                 | 老師建立的練習主題，包含多個句型 | 「老師建立一個主題」 |
| 句型     | Sentence Pattern      | 主題下的語法結構範本             | 「新增多個句型範本」 |
| 作答     | Submission            | 學生提交的練習內容（口說或寫作） | 「學生提交作答內容」 |
| 訂正     | Correction            | 學生根據 AI 建議修改答案的過程   | 「學生可依建議訂正」 |
| 回饋     | Feedback              | AI 針對學生作答給予的錯誤說明    | 「AI 給予錯誤回饋」  |
| 修正建議 | Correction Suggestion | AI 提供的具體改善方向            | 「AI 提供修正建議」  |
| 評分     | Scoring/Grading       | AI 根據正確率計算的分數          | 「系統進行評分」     |
| 文字紀錄 | Text Content          | 學生作答的文字版本（含 AI 轉譯） | 「系統保存文字紀錄」 |

## 作業狀態

| 術語   | 英文      | 定義                         |
| ------ | --------- | ---------------------------- |
| 待處理 | Pending   | 學生已提交但尚未訂正完成     |
| 已訂正 | Corrected | 學生已進行訂正但可能繼續修改 |
| 已完成 | Completed | 學生完成所有訂正，作業結束   |
| 逾期   | Overdue   | 超過截止日期但仍可補交       |

## 練習類型

| 術語     | 英文              | 定義                      |
| -------- | ----------------- | ------------------------- |
| 單句練習 | Sentence Practice | 針對單一句型的練習        |
| 主題對話 | Topic Dialogue    | 基於主題的完整對話練習    |
| 口說模式 | Speaking Mode     | 學生錄音作答，AI 轉譯文字 |
| 寫作模式 | Writing Mode      | 學生直接輸入文字作答      |

## 評分相關

| 術語       | 英文               | 定義                                       |
| ---------- | ------------------ | ------------------------------------------ |
| 初次評分   | Initial Score      | 學生第一次提交時的分數                     |
| 訂正加分   | Correction Bonus   | 學生訂正後獲得的額外分數                   |
| 總分       | Total Score        | 初次評分 + 訂正加分                        |
| 正確單字數 | Correct Word Count | 符合標準答案的單字數量                     |
| 評分策略   | Scoring Strategy   | 老師設定的評分參數（年齡、程度、補充說明） |

## 使用規範

1. **一致性原則**：相同概念必須使用相同術語，避免「訂正」與「修正」混用
2. **精確性原則**：「作答」指學生提交內容，「回饋」指 AI 的錯誤說明
3. **可擴展性**：新增術語時需更新此表並通知團隊
