# 釐清問題

作業達成率與錯誤次數的計算方式為何？

# 定位

Feature：output-training-assign.feature 作業成績與統計 Scenario

# 多選題

| 選項  | 描述                   |
| ----- | ---------------------- |
| A     | 達成率僅看訂正後正確率 |
| B     | 達成率含訂正次數懲罰   |
| C     | 錯誤次數僅計算初次提交 |
| D     | 錯誤次數含所有訂正次數 |
| Short | 其他（請簡述，<=5 字） |

# 影響範圍

影響成績計算、學生作業完成判斷、老師查詢統計。

# 優先級

High

---

# 解決記錄

- **回答**：錯誤次數不用被記錄，但會影響分數。分數計算：單句練習分數 = 正確單字數 ÷ 標準答案單字總數 × 100，語序或語法錯誤會影響正確單字數。主題練習以單句計分邏輯為基礎，逐句計算。
- **更新的規格檔**：spec/features/output-training/output-training-assign.feature
- **變更內容**：補充作業成績與統計情境，明確定義分數計算方式與錯誤次數邏輯。
