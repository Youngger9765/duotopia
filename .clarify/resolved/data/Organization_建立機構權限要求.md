# 釐清問題

誰可以建立機構？

# 定位

Feature: 建立機構
API 權限驗證設計

# 多選題

| 選項 | 描述                                   |
| ---- | -------------------------------------- |
| A    | 僅 Platform Owner（系統管理員）        |
| B    | 任何註冊的教師                          |
| C    | 需要申請並經審核通過                    |
| D    | 付費訂閱者（BASIC 或 PRO）              |

# 影響範圍

- 建立機構 API 的權限驗證邏輯
- 教師註冊流程設計
- 前端 UI 顯示（是否顯示「建立機構」按鈕）
- 機構審核流程設計（若選 C）

# 優先級

Medium

---

# 解決記錄

- **回答**：A - 僅 Platform Owner（系統管理員）
- **更新的規格檔**：spec/features/organization/建立機構.feature
- **變更內容**：新增 Rule「僅系統管理員可以建立機構」，包含 2 個 Example（Platform Owner 成功建立、一般教師被拒絕）
