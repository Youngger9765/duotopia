# 釐清問題

機構擁有人管理流程.feature 說明新增教師時「已使用教師授權數應為 1」，但沒有說明：
- 移除教師時授權數是否釋放？
- 停用教師（is_active=false）是否仍佔用授權？
- 授權數如何計算（COUNT is_active=true？）
- 授權數資料如何儲存（機構表欄位？即時計算？）

# 定位

Feature: 機構擁有人管理流程.feature
規則: 新增機構教師
場景: 新增已註冊的教師到機構
Then: "已使用教師授權數應為 '1'"

ERM: organizations 表（是否需新增 teacher_limit, teacher_used 欄位？）

# 多選題

| 選項 | 描述 |
|------|------|
| A | 停用教師釋放授權（COUNT is_active=true） |
| B | 停用教師仍佔用授權（COUNT all） |
| C | 僅刪除教師才釋放授權 |
| D | 需在 organizations 表新增欄位記錄 |
| E | 即時計算，無需儲存 |

# 影響範圍

- 影響 organizations 表設計（是否需新增欄位）
- 影響教師管理邏輯（新增/移除/停用）
- 影響計費與授權管理
- 需在 DBML 中補充授權數欄位定義
- 需在 Feature 中補充授權釋放 Example

# 優先級

High

- 影響核心業務邏輯（授權管理）
- 影響資料模型設計
- 影響計費策略
