# 釐清問題

建立機構.feature 僅有 2 個 Example（成功建立、一般教師拒絕），缺少以下邊界條件的測試案例：
- 統編重複
- 必填欄位缺失
- 欄位長度超限
- Email 格式驗證

# 定位

Feature: 建立機構.feature
Rule: 僅系統管理員可以建立機構

# 多選題

| 選項 | 描述 |
|------|------|
| A | 補充所有邊界條件 Example |
| B | 邊界條件在單元測試中覆蓋，Feature 保持簡潔 |
| C | 補充部分關鍵邊界條件（統編重複） |
| D | 建立獨立的驗證規則 Rule |
| Short | 提供其他簡短答案（<=5 字） |

# 影響範圍

- 影響測試覆蓋度
- 影響 API 驗證邏輯的明確性
- 影響錯誤訊息設計

High
