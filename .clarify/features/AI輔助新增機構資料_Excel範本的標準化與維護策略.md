# 釐清問題

AI輔助新增機構資料.feature 提到「下載範本」與「智能識別欄位」，但沒有說明：
- Excel 範本由誰維護？如何更新？
- 範本是否支援多語言？
- 智能識別的匹配規則如何定義？
- 若識別錯誤如何處理？

場景: 提供標準 Excel 範本下載、上傳非標準格式的 Excel
