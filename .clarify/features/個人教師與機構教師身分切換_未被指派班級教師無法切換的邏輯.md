# 釐清問題

個人教師與機構教師身分切換.feature 中，「未被指派為班級老師的機構教師查看後台」時「應看不到『切換到機構後台』的選項」，但其他規則顯示切換是基於「機構教師」身份，而非「班級老師」身份，邏輯是否矛盾？

# 定位

Feature: 個人教師與機構教師身分切換.feature
規則: 在個人教師後台查看資料
場景: 未被指派為班級老師的機構教師查看後台
Then: "我應看不到『切換到機構後台』的選項"

vs.

規則: 切換到機構教師身分查看機構資料
場景: 機構教師切換到機構後台（已指派班級）

# 多選題

| 選項 | 描述 |
|------|------|
| A | 必須被指派班級才能切換到機構後台 |
| B | 只要加入機構即可切換，無需班級 |
| C | 根據機構角色決定（org_admin 可切換，teacher 不可） |
| D | 場景描述有誤，應修正 |
| Short | 提供其他簡短答案（<=5 字） |

# 影響範圍

- 影響身份切換邏輯
- 影響 UI 顯示規則
- 影響機構教師的體驗
- 需修正 Feature 中的矛盾描述

# 優先級

High

- 核心功能邏輯矛盾
- 影響使用者體驗
- 需釐清切換條件
