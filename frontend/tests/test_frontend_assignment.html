🧪 前端指派功能測試

測試步驟

重要: 請確保前端開發伺服器正在運行 (http://localhost:5173)

📋 手動測試清單

請按照以下步驟測試指派功能：

  1. 打開作業詳細頁面
  2. 檢查學生列表顯示
    • ✅ 12 個學生都有顯示
    • ✅ 已完成學生顯示「批改」按鈕
    • ✅ 未開始/進行中學生顯示「取消指派」按鈕
  3. 測試指派功能
    • ✅ 勾選未指派的學生
    • ✅ 點擊「指派」按鈕
    • ✅ 確認 API 請求成功 (F12 查看 Network)
    • ✅ 頁面重新整理後學生仍然被指派
  4. 測試取消指派功能
    • ✅ 點擊已指派學生的「取消指派」按鈕
    • ✅ 確認 API 請求成功
    • ✅ 學生狀態正確更新

🔍 檢查重點

API 檢查 (F12 開發者工具 → Network):
  • PATCH /api/teachers/assignments/{id} 返回 200 OK
  • 請求 body 包含正確的 student_ids (數字陣列)
  • response 資料結構正確
前端檢查:
  • 無 console 錯誤
  • 按鈕顯示邏輯正確
  • 狀態更新即時
  • 重刷後資料持久化
