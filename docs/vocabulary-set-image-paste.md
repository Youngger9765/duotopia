# 單字集圖片貼上功能使用指南

本文件說明如何在單字集編輯器中使用剪貼簿貼上圖片功能。

## 功能概述

老師在編輯單字集時，可以直接使用鍵盤快捷鍵貼上螢幕截圖或複製的圖片，無需手動選擇檔案上傳。

## 使用步驟

1. **選擇目標行**：點擊要新增圖片的單字行（任何輸入框或區域皆可）
2. **確認高亮**：該行會顯示淺藍色背景和左側藍色邊條
3. **貼上圖片**：使用鍵盤快捷鍵貼上

---

## macOS 使用方式

### 方式一：截圖直接存入剪貼簿（推薦）

| 快捷鍵 | 說明 |
|--------|------|
| `Ctrl + Shift + Cmd + 3` | 全螢幕截圖存入剪貼簿 |
| `Ctrl + Shift + Cmd + 4` | 區域截圖存入剪貼簿 |
| `Ctrl + Shift + Cmd + 4` 然後按 `Space` | 視窗截圖存入剪貼簿 |

截圖後直接按 `Cmd + V` 即可貼上。

### 方式二：從截圖縮圖複製

1. 使用 `Shift + Cmd + 4` 進行區域截圖
2. 截圖後右下角會出現縮圖預覽
3. 點擊縮圖開啟預覽視窗
4. 按 `Cmd + C` 複製圖片
5. 回到單字集編輯器，按 `Cmd + V` 貼上

### 方式三：複製現有圖片檔案

1. 在 Finder 中找到圖片檔案
2. 選擇檔案後按 `Cmd + C` 複製
3. 回到單字集編輯器，按 `Cmd + V` 貼上

### 方式四：從其他應用程式複製

1. 在瀏覽器、文件或圖片編輯軟體中選擇圖片
2. 按 `Cmd + C` 複製
3. 回到單字集編輯器，按 `Cmd + V` 貼上

### 為什麼 Shift+Cmd+4 截圖後無法直接貼上？

這是 macOS 的設計行為：

| 快捷鍵 | 行為 | 可否貼上 |
|--------|------|----------|
| `Shift + Cmd + 4` | 截圖存檔到桌面 | ❌ 不行 |
| `Ctrl + Shift + Cmd + 4` | 截圖存入剪貼簿 | ✅ 可以 |

`Shift + Cmd + 4` 會將截圖存為檔案，而非存入剪貼簿。因此瀏覽器無法存取該圖片。

## Windows 使用方式

### 方式一：使用截圖工具（推薦）

| 快捷鍵 | 說明 |
|--------|------|
| `Win + Shift + S` | 開啟截圖工具，選擇區域後自動存入剪貼簿 |
| `PrtScn` | 全螢幕截圖存入剪貼簿 |
| `Alt + PrtScn` | 當前視窗截圖存入剪貼簿 |

截圖後直接按 `Ctrl + V` 即可貼上。

### 方式二：使用 Snipping Tool

1. 開啟「剪取工具」(Snipping Tool)
2. 選擇截圖模式並擷取
3. 截圖會自動存入剪貼簿
4. 回到單字集編輯器，按 `Ctrl + V` 貼上

### 方式三：複製現有圖片檔案

1. 在檔案總管中找到圖片檔案
2. 選擇檔案後按 `Ctrl + C` 複製
3. 回到單字集編輯器，按 `Ctrl + V` 貼上

### 方式四：從其他應用程式複製

1. 在瀏覽器、文件或圖片編輯軟體中選擇圖片
2. 按 `Ctrl + C` 複製
3. 回到單字集編輯器，按 `Ctrl + V` 貼上

## 注意事項

### 圖片限制

- **檔案大小**：最大 2MB
- **支援格式**：JPG、PNG、GIF、WebP

### 常見問題

**Q: 貼上後沒有反應？**
- 確認已點擊選擇目標行（該行應顯示藍色高亮）
- 確認剪貼簿中有圖片內容
- macOS 使用者請確認使用 `Ctrl + Shift + Cmd + 4` 截圖

**Q: 該行已有圖片怎麼辦？**
- 系統會提示「此行已有圖片，請先移除」
- 請先點擊現有圖片右上角的 X 按鈕移除後再貼上

**Q: 手機版可以使用嗎？**
- 手機版不支援剪貼簿貼上功能
- 請使用「點擊上傳」按鈕從相簿選擇圖片
